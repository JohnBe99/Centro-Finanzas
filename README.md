# 💰 Centro · Conciencia Financiera

App personal de finanzas para salir de deudas y tomar control de tu dinero.

## Funcionalidades
- 📋 Perfil financiero con ingresos, gastos y deudas
- 🎯 Plan de eliminación de deudas (Método Avalancha)
- 💸 Registro de gastos diarios
- 🤖 Consejero IA personalizado
- 📊 Reporte completo con exportación a Excel
- 🟢🟡🔴 Estrategias: Conservador / Balanceado / Agresivo

## Instalación local

```bash
npm install
npm start
```

## Deploy
Cada push a `main` despliega automáticamente a GitHub Pages.
